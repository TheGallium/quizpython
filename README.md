# ask.question()


### Use an argument to define the displayed question.
```py
ask.question("Hello, how are you ?")
```

```
Hello, how are you ?
 > 
```
### You can use the responce of the user in a variable, here `name`.
```py
name = ask.question("What is your name ?")
print("Hello, " + name)
```
```
What is your name ?
 > Tom Cruise

Hello, Tom Cruise
```
